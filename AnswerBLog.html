Question-1: What are the different ways to select an element in the DOM?

We can select elements using:

  • document.getElementById("id") - Selects an element by its ID.
  • document.getElementsByClassName("class") - Selects elements by class name.
  • document.getElementsByTagName("tag") - Selects elements by tag name.
  • document.querySelector("selector") - Selects the first matching element.
  • document.querySelectorAll("selector") - Selects all matching elements.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: Returns or sets the HTML content of an element.
innerText: Returns or sets only the visible text (ignores hidden elements).
textContent: Returns or sets all text, including hidden elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event triggered on a child element propagates up to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating: document.createElement("tag")
Adding: parent.appendChild(element) or parent.insertBefore(element, reference)
Removing: element.remove() or parent.removeChild(child)
